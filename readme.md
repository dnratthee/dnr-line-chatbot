# DNR Line Chat Bot Test

This is a simple chat bot that I created to test the Line Messaging API. The bot is written in Golang receiving messages from the Line Messaging API and then processes the messages and sends a response back to the Line Messaging API.

## Technologies and Tools

- [Line Messaging API](https://developers.line.biz/en/docs/messaging-api/)
- [Node.js](https://nodejs.org/)
- [Golang](https://golang.org/)
- [Docker](https://www.docker.com/)
